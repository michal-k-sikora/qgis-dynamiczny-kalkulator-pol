about= <ul><li><b>EN: Dynamic Field Calculator</b> enables fast and simultaneous updating of multiple attributes in a selected vector layer, eliminating the need to repeatedly run the standard Field Calculator. The user selects a layer, chooses a set of fields for editing, and defines independent QGIS expressions for each of them within a single panel. The plugin also allows saving field and expression configurations, which streamlines repetitive operations in spatial data analysis and processing workflows.</li><p><li><b>PL: Dynamiczny Kalkulator Pól</b> umożliwia szybkie i jednoczesne aktualizowanie wielu atrybutów w wybranej warstwie wektorowej, z pominięciem konieczności wielokrotnego uruchamiania standardowego Kalkulatora pól. Użytkownik wskazuje warstwę, wybiera zestaw pól do edycji i w jednym panelu definiuje dla każdego z nich niezależne wyrażenia QGIS. Wtyczka umożliwia także zapamiętywanie konfiguracji pól i wyrażeń, co usprawnia wykonywanie powtarzalnych operacji w procesach analizy i przetwarzania danych przestrzennych.</li></p></ul>
